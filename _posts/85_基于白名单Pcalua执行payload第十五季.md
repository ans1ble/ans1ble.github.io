### Pcalua简介：

Windows进程兼容性助理(Program Compatibility Assistant)的一个组件。

**说明：**Pcalua.exe所在路径已被系统添加PATH环境变量中，因此，Pcalua命令可识别

Windows 7 默认位置：  
```bash
C:\Windows\System32\pcalua.exe
```

**攻击机：** 192.168.1.4 Debian  
**靶机：** 192.168.1.5 Windows 7
